### **PYTHON SERVICES (42 Services)**

#### **Build Process:**
```bash
# 1. Navigate to service directory
cd backend-services/python-services/{service-name}

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run tests
python -m pytest tests/ -v

# 4. Build Docker image
docker build -t gogidix/{service-name}:latest .

# 5. Run locally for testing
uvicorn main:app --host 0.0.0.0 --port {port} --reload

# 6. Production deployment
docker run -d --name {service-name} -p {port}:{port} gogidix/{service-name}:latest
```

#### **Testing Procedure:**
```bash
# Unit Tests
python -m pytest tests/unit/ -v --cov=.

# Integration Tests
python -m pytest tests/integration/ -v

# API Tests
python -m pytest tests/api/ -v

# Load Tests
locust -f tests/locustfile.py --host=http://localhost:{port}

# Health Check
curl http://localhost:{port}/health
```
